Proceso de registro:

-   Anime Frontend proveerá una pagina para registrarse de forma normal
-   Una vez hecho el registro se le otorgará a cada usuario el rol de Anime Watcher o Adult Anime Watcher
-   Dentro de la pagina, hay que checkear si el usuario posee alguno de esos roles
-   En caso de que no posea alguno de esos 2 roles, redirigir a 403
-   Si posee los roles, se redirige al usuario al Home Page
